# RecipeSage Automatic Recipe Clipper

An extension for clipping recipes to your RecipeSage account.

## Features

- **One-Click Clipping:** Automatically extracts recipe information (ingredients, instructions, images, etc.) from supported websites.
- **Interactive Clipper:** Manually select fields or refine auto-detected content with a hardware-accelerated, draggable UI.
- **Enhanced Hebrew Support:** Specialized parsing logic to automatically detect and extract recipe content from Hebrew websites, bypassing English-centric ML limitations.
- **Clean & Fast:** Highly optimized for performance with smooth UI transitions.

## Installation (Development Mode)

1.  Clone this repository.
2.  Open Chrome and navigate to `chrome://extensions`.
3.  Enable **Developer mode** in the top right.
4.  Click **Load unpacked** and select the `extension_src` directory within this repository.
5.  Pin the RecipeSage icon to your toolbar for easy access.

## Usage

1.  Navigate to any recipe website.
2.  Click the RecipeSage icon.
3.  Choose **Auto Import** for instant results or **Interactive Import** to refine the clipping yourself.
